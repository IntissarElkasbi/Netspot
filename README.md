# Netspot
Dans ce projet, j’ai utilisé le logiciel NetSpot pour analyser et améliorer la couverture Wi-Fi dans un bâtiment composé de plusieurs salles de TP. L’analyse m’a montré que certaines zones, comme S_TP4 et S_TP6, avaient une couverture faible ou instable. Pour résoudre ce problème, j’ai ajouté un nouveau point d’accès dans une zone stratégique, éloignée du point ENSATE-WIFI, afin d’éviter les interférences entre les deux signaux. Lors de la planification, j’ai également pris en compte le type de mur (brique, béton, etc.), car cela influence la propagation du signal. Par exemple, certains matériaux bloquent ou affaiblissent le Wi-Fi. En choisissant bien l’emplacement et en tenant compte des obstacles, j’ai pu améliorer la couverture et assurer une connexion plus stable dans toutes les salles.
